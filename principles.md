# Living Principles

This file is the `.principles` selection file. It references the `intent-book` group, which activates all 54 principles in `principles/iec/`. See `groups/intent-book.yaml` for the full list.

To browse the principles, see the [Living Principles appendix](https://intent-book.dev/appendices/living-principles) or the individual files in `principles/iec/`. To add a new principle, add a file in the appropriate `principles/iec/<category>/` directory and add its ID to `groups/intent-book.yaml`.
