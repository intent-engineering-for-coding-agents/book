# Living Principles

This file is the `.principles` selection file. It references the `ase-book` group, which activates all 54 principles in `principles/ase/`. See `groups/ase-book.yaml` for the full list.

To browse the principles, see the [Living Principles appendix](https://ase-book.dev/appendices/living-principles) or the individual files in `principles/ase/`. To add a new principle, add a file in the appropriate `principles/ase/<category>/` directory and add its ID to `groups/ase-book.yaml`.
