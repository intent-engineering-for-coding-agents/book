# Code of Conduct

## Our pledge

This project welcomes contributions and discussion from everyone. We are committed to a harassment-free, respectful experience for all participants, regardless of background or identity.

## Our standards

We expect participants to be considerate, constructive, and respectful in issues, pull requests, and discussions. Focus on the work, assume good faith, and accept feedback gracefully. Behavior that harasses, demeans, or excludes others is not welcome.

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its full code of conduct. That document defines the expected standards and the responsibilities of maintainers in detail.

## Scope

This Code of Conduct applies in all project spaces, including the repository, issues, pull requests, and discussions, and when an individual represents the project in public spaces.

## Reporting

Report unacceptable behavior to the project maintainer via GitHub at https://github.com/flemming-n-larsen. All reports will be reviewed and handled with discretion. Maintainers are obligated to respect the privacy and security of anyone who reports an incident.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
