# About the Author

Flemming Nørnberg Larsen is a principal software developer based near Aalborg, Denmark. He works at Elsevier, where he builds and maintains software systems and has spent several years working alongside coding agents in production.

He wrote the [Plain Text as Code Manifest](https://github.com/Plain-Text-as-Code) before writing this book. The Foundation topic in this book is its application to agentic engineering.

He also built [.principles](https://github.com/.principles), a catalog of software engineering principles as plain-text files that a coding agent reads at the start of a session. The AI already knows everything. The question is whether it knows what you care about. The book has a chapter on it.

This book started as irritation at advice that skipped the hard parts. It became an attempt to write down what professional, deliberate agent-assisted development actually looks like: the decisions, the constraints, the failures, and the proof that the result holds past Friday.
